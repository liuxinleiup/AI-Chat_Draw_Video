# 二：AI绘画
## 描述词  
大多数的AI绘画都会有描述词，很多还会有描述词的增强和减弱，越是详细生成的图片就会精美，这里提供一些方法  

第一种：赋予AI聊天绘画师角色让它帮我们生成  
第二种：通过上传照片，下面会自动生成叙述，我们再这个基础上进行修改  
第三种：利用别人的描述词，我们再这个基础上进行修改            
第四种：借助工具，例如[AI画廊](https://www.aigallery.top/)  
          
注意：不同平台间相互交叉描述生成会有不一样的效果
